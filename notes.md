# CPU Instruction Set

## Registers

- Registers A-L (8 bits)
- - Register A - Accumulator, used to perform mathematical operations
- - Register F - Flag (i.e. this value is used for conditional jump operations)
- - - (7 6 5 4 3 2 1 0)
- - - (Z N H CY - - - -)
- - - (Zero flag, sub flag, half carry flag, carry flag)
- PC, SP (16 bits)
